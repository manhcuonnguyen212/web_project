## ĐỀ TÀI SỐ 5

## 1. Tên đề tài: Xây dựng website dạng blog
## 2. Số lượng sinh viên yêu cầu: 2-3 sinh viên.
## 3. Mô tả đề tài	
	Các yêu cầu chính của đề tài:
a. Chức năng dành cho người dùng        
- Hiển thị danh sách các các bài viết.
- Phân loại bài viết theo danh mục.
- Hỗ trợ tìm kiếm bài viết theo tiêu đề và nội dung bài viết.
- Hỗ trợ người dùng đăng ký/ đăng nhập tài khoản để có thể thêm bình luận trong bài viết, hỗ trợ người dùng để lại bình luận dưới mỗi bài viết.
- Hỗ trợ người dùng quản lý thông tin tài khoản bao gồm: Xem và cập nhật thông tin liên hệ(tên, năm sinh, địa chỉ, sđt..). Hỗ trợ người dùng thay đổi mật khẩu.
b. Chức năng dành cho quản trị viên        
- Chức năng đăng nhập dành cho quản trị viên
- Quản lý danh sách tài khoản trong hệ thống, hỗ trợ xóa, sửa thông tin
- Quản lý danh sách các bài viết, thêm, sửa, xóa, ẩn, hiện.
- Quản lý danh sách các danh mục bài viết.
- Quản lý tài khoản của quản trị viên (thêm mới, xoá, cập nhật mật khẩu).
## 4. Yêu cầu nhóm và học viên.
Phát triển ứng dụng Web phía frontend dựa trên ReactJS, HTML, CSS để xây dựng các trang Web đáp ứng các yêu cầu về chức năng giao diện trên.[CLO1]
Phát triển ứng dụng Web phía backend dựa trên NodeJS/ExpressJS hoặc các ngôn ngữ khác tương đương để đáp ứng các yêu cầu nghiệp vụ trên. [CLO1]
Hoàn thiện, đóng gói hệ thống để có thể chạy hoàn chỉnh và triển khai trong thực tế.[CLO2]
